# Network Design Document

## Executive Summary

**Project:** Secure Network Infrastructure for Educational Institution  
**Location:** Kabul, Afghanistan  
**Institution:** Qamar Private High School  
**Date:** March 2025  
**Network Engineer:** Abdul Qahar Ranjbar

## 1. Project Overview

### 1.1 Background
3-floor educational facility requiring secure, segmented network for:
- Administration (Floor 1): 4 users, high security
- Teachers (Floor 2): 5 users, medium security  
- Students (Floor 3): 16+ users, restricted access

### 1.2 Objectives
- ✅ Network segmentation via VLANs
- ✅ Inter-VLAN routing with access control
- ✅ Port security on all access ports
- ✅ Internet access with content filtering
- ✅ Centralized management and monitoring
