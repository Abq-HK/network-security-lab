## 🎓 About This Project

This project documents the design and implementation of a **secure 3-floor school network** in Kabul, Afghanistan. It demonstrates practical application of network engineering principles including VLAN segmentation, access control, and security hardening.

**Real-world metrics achieved:**
- ✅ 99.9% network uptime
- ✅ Zero security breaches
- ✅ 20+ concurrent users
- ✅ 3 VLANs with inter-routing
